# DiscordPidorBot

## Как запустить?

1) Создайте своего бота тут: https://discord.com/developers/ и подключите его себе на сервер
2) `npm install`
3) `cp .env.example .env` и заполнить его
4) `npm run start` я бы рекомендовал запускать через `pm2`. Но сами решайте
